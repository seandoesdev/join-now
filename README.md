# join-now
프로젝트 팀원 매칭 사이트

# ✅ Commit Convetion
## 💡 Commit 예시
- `feat: login 기능 구현`

## 💡 Commit 태그

|태그|설명|
|----|-----|
|Feat|새로운 기능을 추가하는 경우|
|Fix|버그를 고친 경우|
|Docs|문서를 수정한 경우|
|Style|코드 포맷 변경, 세미콜론 누락, 코드 수정이 없는경우|
|Refactor|코드 리팩토링|
|Test|테스트 코드. 리팩토링 테스트 코드를 추가했을 때|
|Chore|빌드 업무 수정, 패키지 매니저 수정|
|Design|CSS 등 사용자가 UI 디자인을 변경했을 때|
|Rename|파일명(or 폴더명) 을 수정한 경우|
|Remove|코드(파일) 의 삭제가 있을 때. "Clean", "Eliminate" 를 사용하기도 함|


# ✅ 구현

## 메인 페이지
- 카드 형식의 게시판
- 모든 유저의 모집 공고 표시

## 로그인 페이지
- 일반 로그인
- 소셜 로그인

## 마이페이지
- 회원정보관리
- 모집 혹은 신청 중인 공고 목록 조회
- 팀 페이지 연동
- 공고 지원에 대한 알림 기능

## 모집 공고(게시글)
- 모집 게시글 작성, 수정, 삭제
- 댓글 기능

## 팀 페이지
- 팀 정보
- 일정표
- 회의록
- 팀 & 팀원 관리
